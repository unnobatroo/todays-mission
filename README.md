TODAY'S MISSION.

## Features

- Random task selection from a built-in list.
- Add custom tasks and persist them in localStorage.
- Theme toggle.

## Project Structure

```text
todays-mission/
├── index.html   # Markup and app shell
├── styles.css   # All visual styles and theme variables
└── script.js    # App logic, state handling, and event wiring
```

## Data Storage

The app uses browser localStorage keys:

- `savedActivities`
- `currentActivity`
- `theme`
